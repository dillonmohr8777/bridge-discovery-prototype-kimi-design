# Bridge provisional brand kit

Status: discovery recommendation, not client-approved
Prepared: 2026-07-11

## Recommendation: Trusted Current

Bridge should initially feel like trusted business infrastructure with a human connection layer. This direction avoids both generic corporate-software blue and the cannabis category's overused leaf/bright-green vocabulary.

### Core attributes

- Credible, not clinical
- Modern, not trendy
- Connected, not noisy
- Industry-aware, not stereotypical
- Warm, not casual

### Color system

| Token | Hex | Intended use |
|---|---:|---|
| Bridge Navy | `#12324A` | Logo field, navigation emphasis, primary actions, large branded surfaces |
| Current Teal | `#0A766E` | Links, selection, connection moments, secondary visual identity |
| Signal Amber | `#D9820F` | Introductions, pending states, small high-attention details |
| Canvas | `#F5F8F7` | Page background |
| Surface | `#FFFFFF` | Cards, fields, tables |
| Primary Text | `#14232E` | Headlines and body text |
| Muted Text | `#5D6D76` | Supporting metadata on light surfaces |
| Border | `#CEDBD7` | Dividers and controls |
| Success | `#177245` | Verified and completed states |
| Error | `#B42318` | Blocking or failed states |

Do not use amber for small body text on white. Do not rely on color alone for verification or error states; pair it with a label and icon.

### Typography

Prototype stack: Aptos Display/Segoe UI/system sans. This is intentionally license-free and portable. Before production, Tori should choose between:

- A neutral grotesk for a software-forward identity
- A humanist sans for a warmer professional network
- A restrained display face used only for marketing headlines, paired with a highly legible UI sans

The UI should maintain a minimum 16px body size, short line lengths, clear hierarchy, and WCAG-conscious contrast.

### Mark and wordmark

The provisional mark is a simplified bridge inside a rounded square. It communicates connection directly and scales to an app icon. It is a placeholder to test recognition and layout—not a final trademark-ready logo. Before approval, complete a trademark/domain/social-handle review and commission or refine a production vector identity.

Avoid:

- Cannabis leaves as the primary mark
- Gradients that reduce small-size legibility
- Thin bridge lines that disappear below 24px
- Lockups reading “The Ecosystem,” which is the retired working name

### UI principles

- Profiles should foreground role, location, verification, and partnership intent.
- Contact details remain permissioned; the first action is a structured request.
- Verification labels must communicate status without implying legal endorsement.
- Administrative screens prioritize review clarity, evidence, and an audit trail.
- Marketing surfaces can be expressive; operational screens should remain calm and dense enough for daily use.

## Alternate directions for the meeting

### Modern Network

Ink `#17152D`, electric violet `#6556E8`, coral `#D95848`, soft canvas `#F7F6FB`. Best if Tori wants Bridge to feel like a venture-backed professional network with a broader future beyond cannabis.

### Botanical Ledger

Forest `#173C2C`, sage `#4F6F57`, copper `#B45F2A`, warm canvas `#F6F5EF`. Best if Tori wants a premium, category-aware identity that remains more mature than consumer cannabis branding.
